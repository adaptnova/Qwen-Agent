# Runbooks

This directory contains task-focused runbooks for operating the Qwen-Agent + vLLM stack.

- `coder_endpoint_setup.md` – end-to-end setup guide for provisioning the coder service with native tool calling.
- `coder_endpoint_validation.md` – procedures for validating tool execution via automation and manual checks.
- `coder_endpoint_troubleshooting.md` – quick answers for common failure modes and escalation points.

Keep runbooks concise, actionable, and current with the live deployment configuration. Cross-reference deeper background or architecture notes from the reference section when needed.
